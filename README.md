# 🌿 榕耀管顧 RongRise Consulting

**從人才策略到 AI 落地，驅動永續成長。智慧轉型，創新未來。**

## 關於我們

榕耀管顧由 C.J. Kuo 老師創辦，專注於協助中小型企業在 AI 時代中穩健轉型。
我們提供三大核心服務：**AI 轉型策略**、**人才發展策略**、**ESG 永續諮詢**。

## 技術棧

- **框架**：Next.js 16 (App Router)
- **樣式**：Tailwind CSS 4
- **語言**：TypeScript
- **字體**：Noto Sans TC + Roboto

## 品牌色板

- Primary: `#194A8D` (Deep Blue)
- Secondary: `#0196EC` (Brighter Blue)
- Tertiary: `#FF9800` (Vibrant Orange)
- Success: `#4CAF50` (Sustainable Green)

## 開發

```bash
npm run dev    # 開發伺服器
npm run build  # 正式建構
npm run start  # 啟動正式伺服器
```
